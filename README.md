# Uzay Savaşı

Gemi kontrol ederek düşmanları yok ettiğin bir uzay savaş oyunudur.

## Açıklama

Ekranın altında kontrol ettiğin gemi, yukarıdan gelen uzay gemilerine ateş edersin. Düşman gemilerini vur ve yaşa. Zorluk gittikçe artar, düşman sayısı ve hızı artarken sen de mücadele etmelisin.

## Kullanım

1. Python 3'ü kur
2. Gerekli kütüphaneleri yükle: `pip install pygame`
3. `UzaySavasi.py` dosyasını çalıştır
4. Oyuna başla ve düşmanları yok et

## Kontroller

- **Sol/Sağ Ok**: Gemiyi hareket ettir
- **SPACE**: Ateş et
- **ESC**: Oyundan çık
